## weiblog

### 一、页面流程设计

1. 首页(Twitter)

2. 登录/注册(Tumblr)

3. 个人主页(微博)

4. 信息修改页面

5. 相册页面

### 二、功能

1. 发布微博

2. 发布图片

3. 话题

4. 个人主页背景图

5. 我的收藏/赞/评论过的微博

6. 赞过的图片

7. 注册登录

8. 热门微博

9. 热门图片

10. 修改个人信息

11. 关注用户

### 三、数据库结构

Tables: `user`,`blog`, `image`, `user_user_relationship`, `user_image_relationship`, `user_blog_relationship`, `blog_image_relationship`


### Tech Stack:

HTML5、JavaScript、CSS、PHP

jQuery
